Trackline by Code Contract · BatteryPassDataModel v1.3

🔄 Como funciona Trackline para el Battery Passport

Trackline gestiona todo el ciclo de datos del DPP: solicita los documentos necesarios (internamente o a terceros), extrae y estructura los datos de cada campo, los valida contra el esquema BatteryPassDataModel, y genera el JSON exportable listo para publicar.
